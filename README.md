# Quiz App

## Overview
Quiz App is a simple trivia game that was created as project #2 of the Treehouse iOS Techdegree, the app allows you to answer trivia questions and see whether you got the questions right or wrong. In addition, there is a limit of 15 seconds per question and your final score is displayed at the end.
